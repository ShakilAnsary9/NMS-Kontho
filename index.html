Let's Know about Installation Requirement

To ensure a seamless installation and utilization of this application, your system must meet the following requirements.

64 bit Windows 10, 11, or later
(Windows 10 is the minimum requirement)

Minimum 4 GB Recommended.

100 MB Available.

This application is not optimized for low-end PCs. Currently only available for Windows operating systems. However, the developer is working on making it available for Linux and Mac as soon as possible. If you intend to download the installer or the portable version, please be aware that some antivirus software may flag these versions. To avoid this issue, you can install the application directly from the Microsoft App Store.
